Black and White:

This image here was developed in Adobe PhotoShop. This uses many tools such as shapes, brushes, opacity, levels, and lines. Using only two real colors and shifting between opacities are what make this image. Along with using levels it can give more contrast within the finished piece.








Our Tools:

In this picture its of me enlarging an objects to a ridiculous size compared to what it normally would be. The software used is Adobe PhotoShop using scale, clipping mask, and erasing. Using these tools it allows me to create this effect.








Repair:

Here I had a picture that projected historical value. The aged and cracked image of the person within had become present. Making the image look like new was the true task at hand. Using PhotoShop with clipping mask and the healing tool it allowed me to repair as much as possible without tarnishing the image.








I Glitched:

In this photo I have taken an image from Google and put it through multiple different processing/renders. Pushing it through softwares like Audacity and Adobe PhotoShop. Glitching it through those renders it allows distortion across the entire image and making it look like a mistake, but still being able to see/know what it is.
